## Version 3.0 — simpler setup, more reliable publishing

This is a major update after several years. Version 3 replaces the old Python tool (buster) with **gssg**, so you no longer need Python 2.

### What's new

- **Fewer dependencies** — Node.js and wget only (no Python)
- **More reliable links and images** on your live GitHub Pages site
- **migrate.sh** — automatic upgrade from Classic v2 with backup
- **Plain-English docs** in the `docs/` folder
