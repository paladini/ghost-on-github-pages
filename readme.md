## Quick start

1. Install [requirements](docs/legacy/REQUIREMENTS.md) (Node.js, Python 2, pip2, buster)
2. Run `./install.sh`
3. Edit at `http://localhost:2373/ghost`
4. Publish with `cd ~/.ghost && ./deploy.sh`
